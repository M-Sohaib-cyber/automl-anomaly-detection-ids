# Dataset Information

This folder contains sample test datasets for evaluating the deployed anomaly detection system.

## Included Files

- `unsw_test.csv` – Sample test data from UNSW-NB15 dataset  

---

## ⚠️ Important Notes

- These are **test subsets only**, not full datasets  
- The deployed application is designed for **UNSW-NB15 raw feature format**  

---

## How to Use

1. Load `unsw_test.csv` into the frontend  
2. Ensure input contains raw features (no preprocessing)  
3. Run predictions
